02 Show Posts Dynamically

We can dynamically load in the JS file and the Custom HTML tags.

This could also be done programatically via the Intersection Observer API - lazy loading of contentthat is below the fold.

In DEV TOOLS, we can see dynamic.js being loaded when button is clicked and in XHR tab we can see the Web Components firing off an AJAX request for each postID.
